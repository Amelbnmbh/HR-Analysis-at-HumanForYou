# HR Analysis at HumanForYou

## Overview
HumanForYou, an Indian company specializing in pharmaceutical products with approximately 4,000 employees, is facing an annual turnover rate of 15%. This turnover presents several challenges for the company:

- **Project Impact**: The departure of key employees slows down projects, affecting the company's reputation with clients and partners.
- **Strain on Human Resources**: Continuous recruitment heavily burdens the HR teams.
- **Loss of Productivity**: The onboarding and training of new employees temporarily reduce productivity.

## Objective
The goal of this project is to identify the key factors influencing the turnover rate through an in-depth data analysis. Using predictive models, the project aims to:

- Understand the root causes of turnover.
- Suggest solutions to reduce turnover and improve employee retention.

## Data
The data is sourced from the Human Resources department and provided in anonymized CSV files. Each employee is represented by a unique identifier (EmployeeID), allowing for an analysis of their career progression within the company.

## Approach
1. **Data Preparation**: Clean and transform the data to make it usable.
2. **Data Exploration**: Conduct exploratory analysis to detect trends and correlations.
3. **Predictive Modeling**: Apply machine learning techniques to predict employee departures.
4. **Recommendations**: Provide actionable suggestions to improve retention.
